## –°–ª–∞–π–¥–µ—Ä—ã

[–î–æ–∫—É–º–µ–Ω—Ç —Å–∫–∞—á–∞—Ç—å –º–æ–∂–Ω–æ –∑–¥–µ—Å—å](./Matplotlib___—Å–ª–∞–π–¥–µ—Ä—ã__3d.pdf)

–ü–æ–Ω—è—Ç–∏—è:
- –°–ª–∞–π–¥–µ—Ä—ã - –®—Ç—É–∫–∏ —Å –ø–æ–º–æ—â—å—é –∫–æ—Ç–æ—Ä—ã—Ö –º–æ–∂–Ω–æ —Ä–µ–≥—É–ª–∏—Ä–æ–≤–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö.
- –§–∏–≥—É—Ä—ã (Figures) - –≤–Ω—É—Ç—Ä–∏ MatPlotLib - **–æ–∫–Ω–∞**, –≤ –∫–æ—Ç–æ—Ä—ã—Ö –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç —Ä–∏—Å–æ–≤–∞–Ω–∏–µ.
- –û—Å–∏ (Axes) - –≤–Ω—É—Ç—Ä–∏ MatPlotLib - –ø–æ–¥–≥—Ä–∞—Ñ–∏–∫–∏, –≤–Ω—É—Ç—Ä–∏ –æ–¥–Ω–æ–≥–æ –æ–∫–Ω–∞ (—Ñ–∏–≥—É—Ä—ã) –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –æ—Å–µ–π, –≤ –∫–∞–∂–¥–æ–º —Å–≤–æ–π –≥—Ä–∞—Ñ–∏–∫.

–ò–º–ø–æ—Ä—Ç–∏—Ä—É–µ–º –±–∏–±–ª–∏–æ—Ç–µ–∫–∏:

```python
import numpy as np
import matplotlib.pyplot as plt
from matplotlib.widgets import Slider
```

–ó–∞–¥–∞–µ–º –Ω–∞—á–∞–ª—å–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è:

```python
x = np.linspace(0, 10, 100) # —Å—Ç–æ —Ç–æ—á–µ–∫ –æ—Ç 0 –¥–æ 10

a = 0
b = 1
c = 1
d = 0

y = a + b * np.sin(c * x + d) # —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä—É—é –±—É–¥–µ–º —Ä–∏—Å–æ–≤–∞—Ç—å
```

–ù–∞—Å—Ç—Ä–∞–∏–≤–∞–µ–º –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ –≥—Ä–∞—Ñ–∏–∫–∞:

```python
fig, ax = plt.subplots() # –°–æ–∑–¥–∞–µ–º –Ω–æ–≤—É—é —Ñ–∏–≥—É—Ä—É –∏ –æ—Å–∏
plt.subplots_adjust(bottom=0.5) # –ó–∞–¥–∞–µ–º –æ—Ç—Å—Ç—É–ø —Å–Ω–∏–∑—É, –æ—Å—Ç–∞–≤–ª—è—è –º–µ—Å—Ç–æ –¥–ª—è —Å–ª–∞–π–¥–µ—Ä–æ–≤
(line,) = ax.plot(x, y) # –†–∏—Å—É–µ–º —Ñ—É–Ω–∫—Ü–∏—é –∏ —Å–æ—Ö—Ä–∞–Ω—è–µ–º –ø–æ–ª—É—á–∏–≤—à—É—é—Å—è –ª–∏–Ω–∏—é
```

–°–æ–∑–¥–∞–µ–º —Å–ª–∞–π–¥–µ—Ä—ã –Ω–∞ –∫–∞–∂–¥—ã–π –ø–∞—Ä–∞–º–µ—Ç—Ä:

```python
# –û—Ç—Å—Ç—É–ø–°–ª–µ–≤–∞, –û—Ç—Å—Ç—É–ø–°–Ω–∏–∑—É, –®–∏—Ä–∏–Ω–∞, –í—ã—Å–æ—Ç–∞
ax_a = plt.axes([0.1, 0.25, 0.65, 0.03])
# –°–ª–∞–π–¥–µ—Ä –ø—Ä–∏–Ω–∏–º–∞–µ—Ç: –æ—Å—å, –Ω–∞–∑–≤–∞–Ω–∏–µ, –Ω–∞—á–∞–ª–æ, –∫–æ–Ω–µ—Ü, –Ω–∞—á–∞–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ
a_slider = Slider(ax_a, 'a', 0.1, 10.0, valinit=a)

ax_b = plt.axes([0.1, 0.20, 0.65, 0.03])
b_slider = Slider(ax_b, 'b', 0.1, 10.0, valinit=b)

ax_c = plt.axes([0.1, 0.15, 0.65, 0.03])
c_slider = Slider(ax_c, 'c', 0.1, 10.0, valinit=c)

ax_d = plt.axes([0.1, 0.1, 0.65, 0.03])
d_slider = Slider(ax_d, 'd', 0.1, 10.0, valinit=d)
```

–°–æ–∑–¥–∞–µ–º —Ñ—É–Ω–∫—Ü–∏—é, –∫–æ—Ç–æ—Ä–∞—è –±—É–¥–µ—Ç –≤—ã–∑—ã–≤–∞—Ç—å—Å—è –∫–∞–∂–¥—ã–π —Ä–∞–∑, –∫–æ–≥–¥–∞ –º—ã –¥–≤–∏–≥–∞–µ–º —Å–ª–∞–π–¥–µ—Ä:

```python
def update(val):
    # –ë–µ—Ä–µ–º —Ç–µ–∫—É—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∫–∞–∂–¥–æ–≥–æ –∏–∑ —Å–ª–∞–π–¥–µ—Ä–æ–≤
    a = a_slider.val
    b = b_slider.val
    c = c_slider.val
    d = d_slider.val

    # –°–Ω–æ–≤–∞ –≤—ã—á–∏—Å–ª—è–µ–º —Ñ—É–Ω–∫—Ü–∏—é, –Ω–æ —É–∂–µ —Å –Ω–æ–≤—ã–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏
    data = a + b * np.sin(c * x + d)
    # –°–Ω–æ–≤–∞ —Ä–∏—Å—É–µ–º —Ñ—É–Ω–∫—Ü–∏—é
    line.set_ydata(data)
    # –ì–æ–≤–æ—Ä–∏–º matplotlib —á—Ç–æ –≥—Ä–∞—Ñ–∏–∫ –∏–∑–º–µ–Ω–∏–ª—Å—è
    fig.canvas.draw_idle()
```

–ü—Ä–∏–≤—è–∑—ã–≤–∞–µ–º —Ñ—É–Ω–∫—Ü–∏—é –∫ —Å–ª–∞–π–¥–µ—Ä–∞–º:

```python
a_slider.on_changed(update)
b_slider.on_changed(update)
c_slider.on_changed(update)
d_slider.on_changed(update)
```

–ü–æ–∫–∞–∑—ã–≤–∞–µ–º –≥—Ä–∞—Ñ–∏–∫:

```python
plt.show()
```

### –ó–∞–¥–∞–Ω–∏–µ

–ù–∞—Ä–∏—Å–æ–≤–∞—Ç—å –≥—Ä–∞—Ñ–∏–∫ —Ñ—É–Ω–∫—Ü–∏–∏ `np.cos(a * x + b) + np.sin(c * x + d)`. –í—Å–µ 4 –ø–∞—Ä–∞–º–µ—Ç—Ä–∞ (a, b, c, d) –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —Ä–µ–≥—É–ª–∏—Ä—É–µ–º—ã.

## 3D ü§Ø

–°–æ–∑–¥–∞–µ–º —Å–µ—Ç–∫—É –¥–ª—è –¥–∞–Ω–Ω—ã—Ö:

```python
X = np.linspace(-1, 1, 1000) # 1000 —Ç–æ—á–µ–∫ –æ—Ç -1 –¥–æ 1
Z = np.linspace(-1, 1, 1000)
(X, Z) = np.meshgrid(X, Z) # –ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω–∞—è —Å–µ—Ç–∫–∞ 1000 –Ω–∞ 1000 —Ç–æ—á–µ–∫ –æ—Ç -1 –¥–æ 1
```

–ó–∞–¥–∞–µ–º —Ñ—É–Ω–∫—Ü–∏—é $y = \pm \sqrt{1 - x^2}$ (Y –¥–ª—è –ø–ª—é—Å, Y2 –¥–ª—è –º–∏–Ω—É—Å):

```python
r = 1 # —Ä–∞–¥–∏—É—Å
Y = np.sqrt(r ** 2 - X ** 2)
Y2 = -Y
```

–°–æ–∑–¥–∞–µ–º –Ω–æ–≤–æ–µ 3d –æ–∫–Ω–æ:

```python
fig = plt.figure(figsize=(10, 7))
ax = fig.add_subplot(111, projection='3d')
```

–†–∏—Å—É–µ–º **–ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–∏** (–≤ 2–¥ –±—ã–ª–∏ –ª–∏–Ω–∏–∏, –∑–¥–µ—Å—å - –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–∏) –≤ –¥–≤–∞ –ø–æ–¥—Ö–æ–¥–∞, —Å–Ω–∞—á–∞–ª–∞ –¥–ª—è $y \geq 0$, –∑–∞—Ç–µ–º –¥–ª—è $y \leq 0$:

```python
surf = ax.plot_surface(X, Y, Z, cmap='viridis')
surf = ax.plot_surface(X, Y2, Z, cmap='viridis')
```

–ó–∞–¥–∞–µ–º –Ω–∞–∑–≤–∞–Ω–∏—è –æ—Å–µ–π:

```python
ax.set_xlabel('X')
ax.set_ylabel('Y')
ax.set_zlabel('Z')
```

–ü–æ–∫–∞–∑—ã–≤–∞–µ–º –≥—Ä–∞—Ñ–∏–∫:

```python

```
